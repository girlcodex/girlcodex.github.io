Key to Production Quality

Symbolic Interactionism, the study of how humans are taught to interface with reality, is the key to Sara's success in UX Design. If users react in predictable ways to controlled stimuli, then user behavior can be modified by modulating that stimuli in repeatable ways. Sara creates archetype-based user mythologies which help define how users react to the real (or implied) presence of entropic state-changes.
